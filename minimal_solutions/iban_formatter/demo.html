Enter an IBAN to format and validate.

API Documentation

Nutze diese Lösung als externe Schnittstelle in deinen eigenen Applikationen.

Endpoint: POST /api/minimal-solutions/iban_formatter
Request Example: 
{
  "iban": "de89370400440532013000"
}
Response Example: 
{
  "status": "success",
  "data": {
    "iban": "DE89370400440532013000",
    "formatted_iban": "DE89 3704 0044 0532 0130 00",
    "is_valid": true
  }
}
