API Integration

Use this interface externally to generate a role-permission access matrix.

Endpoint

POST /api/minimal-solutions/access_role_matrix_demo

Request Example

{
  "roles": ["Admin", "Editor", "Viewer"],
  "permissions": ["read", "write", "delete"]
}

Response Example

{
  "status": "success",
  "data": {
    "matrix": [
      {
        "role": "Admin",
        "permissions": {
          "read": true,
          "write": true,
          "delete": true
        }
      },
      {
        "role": "Editor",
        "permissions": {
          "read": true,
          "write": true,
          "delete": false
        }
      }
    ]
  }
}
