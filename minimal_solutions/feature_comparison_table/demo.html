Configure features and items, then click generate.

API Integration

Use this minimal solution via our REST API in your own applications.

Endpoint: POST /api/minimal-solutions/feature_comparison_table
Request Example (JSON): 
{
  "features": [
    "Storage",
    "Users"
  ],
  "items": [
    {
      "name": "Basic",
      "values": {
        "Storage": "10 GB",
        "Users": "1"
      }
    }
  ]
}
Response Example (JSON): 
{
  "status": "success",
  "data": {
    "features": [
      "Storage",
      "Users"
    ],
    "items": [
      {
        "name": "Basic",
        "values": {
          "Storage": "10 GB",
          "Users": "1"
        }
      }
    ]
  }
}
