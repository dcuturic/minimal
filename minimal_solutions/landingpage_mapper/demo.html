API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um Landingpage-Daten programmgesteuert zu mappen.
        


        

            Endpunkt
            POST /api/minimal-solutions/landingpage_mapper
        


        

            

                Request-Beispiel
                
{
  "topic": "Produkt-Launch",
  "target": "Neukunden",
  "options": {
    "strategy": "direct",
    "include_metadata": true
  }
}

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "topic": "Produkt-Launch",
    "target": "Neukunden",
    "options": {
      "strategy": "direct",
      "include_metadata": true
    },
    "mapper_id": "map_123456",
    "timestamp": "2026-05-13T12:00:00Z",
    "mapped": true
  }
}
