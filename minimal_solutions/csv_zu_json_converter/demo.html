API Documentation

        
Nutze diese Lösung als externe Schnittstelle in deinen eigenen Applikationen.

        
        

            Endpoint: POST /api/minimal-solutions/csv_zu_json_converter
        

        
        

            

                Request Example:
                
{
  "csv_text": "id,name\n1,Alice\n2,Bob",
  "delimiter": ",",
  "has_header": true
}

            

            

                Response Example:
                
{
  "status": "success",
  "data": {
    "json_data": [
      { "id": "1", "name": "Alice" },
      { "id": "2", "name": "Bob" }
    ]
  }
}
