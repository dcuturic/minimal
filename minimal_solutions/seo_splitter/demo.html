API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um SEO-Daten basierend auf Topic, Target und Optionen aufzuteilen.

Endpunkt POST /api/minimal-solutions/seo_splitter
Request-Beispiel 
{
  "topic": "Content Marketing",
  "target": "Beginners",
  "options": {
    "split_by": "paragraphs"
  }
}

Die Eingabeparameter für die SEO Aufteilung.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "result": "..."
  }
}
