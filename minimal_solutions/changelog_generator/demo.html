API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert Changelogs aus Version, Datum und Änderungen zu generieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/changelog_generator
        


        

            

                Request-Beispiel
                
{
  "version": "1.0.0",
  "date": "2026-05-12",
  "changes": [
    "Added new feature X",
    "Fixed bug Y"
  ]
}

                
Hinweis: `date` ist optional.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "result": "## [1.0.0] - 2026-05-12\n\n- Added new feature X\n- Fixed bug Y"
  }
}
