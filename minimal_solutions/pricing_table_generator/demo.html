Configure plans and click generate.

API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Pricing Tabellen-Strukturen zu validieren oder zu generieren.

Endpunkt POST /api/minimal-solutions/pricing_table_generator
Request-Beispiel 
{
  "plans": [
    {
      "name": "Basic",
      "price": "9",
      "currency": "$",
      "period": "/mo",
      "features": ["1 User"],
      "is_popular": false
    }
  ]
}
Response-Beispiel 
{
  "status": "success",
  "data": {
    "result": [
      {
        "name": "Basic",
        "price": "9",
        "currency": "$",
        "period": "/mo",
        "features": ["1 User"],
        "is_popular": false
      }
    ]
  }
}
