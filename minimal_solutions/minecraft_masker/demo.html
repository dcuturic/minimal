Enter input text and mode to mask Minecraft data

Masked Result

API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Minecraft-Daten basierend auf input_text, mode und weiteren Optionen zu maskieren.

Endpunkt POST /api/minimal-solutions/minecraft_masker
Request-Beispiel 
{
  "input_text": "Player Notch just mined 10 diamonds.",
  "mode": "strict",
  "options": {
    "mask_char": "*"
  }
}

Die Felder input_text, mode und options sind optional.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "masked_text": "Player N***h just mined 10 d******s.",
    "mode": "strict",
    "timestamp": "2024-05-14T12:00:00Z"
  }
}
