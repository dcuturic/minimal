API Integration

You can use this tool via API. Send a POST request to /api/minimal-solutions/webhook_payload_tester.

Request Example

{
  "payload_json": "{\n  \"event\": \"user.created\",\n  \"data\": {\n    \"user_id\": \"123\"\n  }\n}"
}

Response Example

{
  "status": "success",
  "data": {
    "is_valid": true,
    "message": "The webhook payload is valid and correctly formatted.",
    "parsed_payload": {
      "event": "user.created",
      "data": {
        "user_id": "123"
      }
    }
  }
}
