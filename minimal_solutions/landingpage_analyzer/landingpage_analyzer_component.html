API Integration

You can use this tool via our REST API. Send a POST request with the topic, target, and options to analyze the data.

Endpoint POST /api/minimal-solutions/landingpage_analyzer

{
  "topic": "Modern AI SaaS",
  "target": "Developers and Tech Founders",
  "options": "Analyze SEO, performance, UX patterns"
}
