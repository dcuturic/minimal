Fülle das Formular aus und klicke auf generieren.

API Dokumentation

Diese Minimal-Lösung ist auch als REST-API verfügbar. Sende einen POST-Request an den unten stehenden Endpunkt.

Endpunkt

POST /api/minimal-solutions/kontaktkarten_generator

Request (JSON)

{
  "name": "Max Mustermann",
  "email": "max@example.com",
  "phone": "+49 123 456789",
  "company": "Muster GmbH",
  "website": "https://example.com"
}

Response (JSON)

{
  "status": "success",
  "data": {
    "vcard": "BEGIN:VCARD...",
    "json": {
      "name": "Max Mustermann",
      "email": "max@example.com",
      "phone": "+49 123 456789",
      "company": "Muster GmbH",
      "website": "https://example.com"
    }
  }
}
