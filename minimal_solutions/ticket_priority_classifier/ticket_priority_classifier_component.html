API Documentation

You can also use this tool via our REST API. Send a POST request to the endpoint below.

Endpoint POST /api/minimal-solutions/ticket_priority_classifier
Example Request 
{
  "ticket_text": "The entire server is down and clients are complaining. Fix immediately!"
}
Example Response 
{
  "status": "success",
  "data": {
    "priority": "Urgent",
    "confidence": 0.98,
    "reason": "Contains terms like 'server is down', 'clients complaining', 'immediately'."
  }
}
