API Dokumentation

Du kannst diese Funktion auch über unsere REST-API nutzen. Sende dazu einen POST-Request an den unten stehenden Endpunkt.

POST /api/minimal-solutions/regex_tester

Request Beispiel

{
  "pattern": "^[a-z]+$",
  "flags": "i",
  "text": "Hello World"
}

Response Beispiel (Erfolg)

{
  "status": "success",
  "data": {
    "matches": [
      {
        "match": "Hello",
        "start": 0,
        "end": 5,
        "groups": []
      },
      {
        "match": "World",
        "start": 6,
        "end": 11,
        "groups": []
      }
    ],
    "count": 2
  }
}
