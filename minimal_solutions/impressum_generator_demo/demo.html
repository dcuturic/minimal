Fülle das Formular aus und klicke auf "Impressum generieren".

Hinweis: Dieses Dokument ist ein automatisch generierter Entwurf und stellt keine Rechtsberatung dar. 

                

            

            
            

                
            

        

    


    

    
    

        
API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert ein Impressum basierend auf den übergebenen Unternehmensdaten zu generieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/impressum_generator_demo
        


        

            

                Request-Beispiel
                
{
  "company": "Muster GmbH",
  "address": "Musterstraße 1, 12345 Musterstadt",
  "email": "info@mustergmbh.de",
  "phone": "+49 123 456789",
  "representative": "Max Mustermann"
}

                
Hinweis: `phone` ist optional, die restlichen Felder sind Pflichtfelder.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "impressum_text": "Impressum\n\nMuster GmbH\nMusterstraße 1, 12345 Musterstadt\n..."
  }
}
