API Documentation

You can use this tool programmatically via our API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/altersrechner

Request Example

fetch('/api/minimal-solutions/altersrechner', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        "birth_date": "1990-05-15",
        "target_date": "2026-05-11"
    })
});
