Configure parameters and click generate.

API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Landingpage-Section-Daten (JSON) generieren zu lassen.

Endpunkt POST /api/minimal-solutions/landingpage_section_generator
Request-Beispiel 
{
  "section_type": "hero",
  "topic": "AI Content Generator"
}
Response-Beispiel 
{
  "status": "success",
  "data": {
    "result": {
      "type": "hero",
      "headline": "Supercharge your AI Content Generator",
      "subheadline": "The best way to manage your AI Content Generator workflow efficiently and effortlessly.",
      "call_to_action": "Get Started Now"
    }
  }
}
