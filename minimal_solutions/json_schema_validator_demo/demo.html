Enter Schema and JSON Data to validate.

    API Integration

    You can use the JSON Schema Validator via API. Send a POST request to the endpoint below.

    POST /api/minimal-solutions/json_schema_validator_demo

    Request Example

    {
  "json_text": {
    "name": "John Doe",
    "age": 30
  },
  "schema_text": {
    "$schema": "http://json-schema.org/draft-07/schema#",
    "type": "object",
    "properties": {
      "name": { "type": "string" },
      "age": { "type": "integer", "minimum": 0 }
    },
    "required": ["name", "age"]
  }
}

    Response Example (Valid)

    {
  "status": "success",
  "message": "JSON Schema validated successfully",
  "data": {
    "is_valid": true,
    "errors": []
  }
}

    Response Example (Invalid)

    {
  "status": "success",
  "message": "JSON Schema validated successfully",
  "data": {
    "is_valid": false,
    "errors": [
      "'age' is a required property"
    ]
  }
}
