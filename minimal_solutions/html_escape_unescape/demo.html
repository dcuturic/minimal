API Schnittstelle

Diese Minimal-Lösung bietet eine REST-API, die extern verwendet werden kann. Sende einen POST-Request mit einem JSON-Body an den Endpunkt, um Text zu escapen oder zu unescapen.

Endpunkt:
POST /api/minimal-solutions/html_escape_unescape
Request Beispiel (JSON): 
{
  "html_text": "<div class=\"test\">Hello World!</div>",
  "mode": "escape"
}
Response Beispiel (JSON): 
{
  "status": "success",
  "data": {
    "result": "&lt;div class=&quot;test&quot;&gt;Hello World!&lt;/div&gt;"
  }
}
