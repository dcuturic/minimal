API Dokumentation

            
Endpoint: POST /api/minimal-solutions/hosting_validator

            
Die Schnittstelle kann von externen Systemen aufgerufen werden, indem ein POST-Request mit den entsprechenden Parametern (source, config, mode) als JSON-Body gesendet wird.

            
Request-Beispiel:

            
{
  "source": "{\"host\": \"example.com\", \"ip\": \"192.168.1.1\"}",
  "config": "{}",
  "mode": "standard"
}
            

            
Response-Beispiel:

            
{
  "success": true,
  "data": {
    "source": "{\"host\": \"example.com\", \"ip\": \"192.168.1.1\"}",
    "config": "{}",
    "mode": "standard",
    "validated_hosting": true,
    "message": "Hosting successfully validated."
  },
  "error": null,
  "warnings": [],
  "meta": {}
}
