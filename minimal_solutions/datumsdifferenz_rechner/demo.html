API Documentation

You can use this tool programmatically via our API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/datumsdifferenz_rechner

Request Example

fetch('/api/minimal-solutions/datumsdifferenz_rechner', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        "start_date": "2023-01-01",
        "end_date": "2023-12-31"
    })
});
