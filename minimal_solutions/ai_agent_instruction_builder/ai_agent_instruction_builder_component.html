API Integration

You can use this tool via our REST API. Send a POST request with the agent_name, purpose, tools, and constraints.

Endpoint POST /api/minimal-solutions/ai_agent_instruction_builder
Request Example 
{
  "agent_name": "Support Bot",
  "purpose": "Answer customer queries",
  "tools": "search_db, get_weather",
  "constraints": "Be polite, do not answer non-support questions"
}
Response Example 
{
  "status": "success",
  "data": {
    "instruction": "You are Support Bot. Your primary purpose is to Answer customer queries...\n\nTools available:\n- search_db\n- get_weather\n\nConstraints:\n- Be polite, do not answer non-support questions"
  }
}
