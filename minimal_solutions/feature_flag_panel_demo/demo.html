API Integration

Use this interface externally to validate and parse feature flags.

Endpoint

POST /api/minimal-solutions/feature_flag_panel_demo

Request Example

{
  "flags": [
    "enable_new_dashboard",
    "beta_search_v2",
    "experimental_dark_mode"
  ]
}

Response Example

{
  "status": "success",
  "data": {
    "flags": [
      {
        "name": "enable_new_dashboard",
        "enabled": false
      },
      {
        "name": "beta_search_v2",
        "enabled": false
      },
      {
        "name": "experimental_dark_mode",
        "enabled": false
      }
    ]
  }
}
