Hosting Diff Viewer

Hosting Diff Viewer zeigt Unterschiede für Hosting-Daten als kleine UI-Demo und stellt dieselbe Funktion als API-Schnittstelle bereit.

Live-Vorschau (Source):
Bitte geben Sie die zu vergleichenden Daten ein und klicken Sie auf "Diff anzeigen".

API Dokumentation

            
Endpoint: POST /api/minimal-solutions/hosting_diff_viewer

            
Die Schnittstelle kann von externen Systemen aufgerufen werden, indem ein POST-Request mit den entsprechenden Parametern (source, config, mode) als JSON-Body gesendet wird.

            
Request-Beispiel:

            
{
  "source": "{\"old_hosting\": {\"provider\": \"aws\"}, \"new_hosting\": {\"provider\": \"gcp\"}}",
  "config": "{\"ignore_fields\": [\"id\"]}",
  "mode": "default"
}
            

            
Response-Beispiel:

            
{
  "success": true,
  "data": {
    "source": "...",
    "config": "...",
    "mode": "default",
    "diff": [
      {"field": "provider", "old": "aws", "new": "gcp"}
    ],
    "status": "success",
    "message": "Diff erfolgreich berechnet."
  },
  "error": null,
  "warnings": [],
  "meta": {}
}
