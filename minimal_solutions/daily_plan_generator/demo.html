API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um programmgesteuert Aufgaben in einen tagesaktuellen, stundenbasierten Plan zu verteilen.

Endpunkt POST /api/minimal-solutions/daily_plan_generator
Request-Beispiel 
{
  "tasks": [
    "Mails checken",
    "Meeting vorbereiten",
    "Ticket 123 bearbeiten"
  ],
  "available_hours": 8
}

Hinweis: `tasks` ist ein Array von Strings, `available_hours` eine Zahl größer 0.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "plan": [
      {
        "task": "Mails checken",
        "start_time": "09:00",
        "end_time": "11:40"
      },
      {
        "task": "Meeting vorbereiten",
        "start_time": "11:40",
        "end_time": "14:20"
      },
      {
        "task": "Ticket 123 bearbeiten",
        "start_time": "14:20",
        "end_time": "17:00"
      }
    ]
  }
}
