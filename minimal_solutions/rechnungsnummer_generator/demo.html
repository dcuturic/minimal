Configure parameters and click generate.

Pattern: [Prefix]-[Year]-[Number]

API Documentation

Use this endpoint to programmatically generate invoice numbers.

POST /api/minimal-solutions/rechnungsnummer_generator

Request Example

{
  "prefix": "RE",
  "year": "2026",
  "start_number": 1,
  "count": 5
}

Response Example

{
  "status": "success",
  "data": {
    "numbers": [
      "RE-2026-0001",
      "RE-2026-0002",
      "RE-2026-0003",
      "RE-2026-0004",
      "RE-2026-0005"
    ],
    "count": 5,
    "pattern": "RE-2026-0000"
  }
}
