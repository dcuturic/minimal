API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um Landingpage-Daten in Segmente aufzuteilen.
        


        

            Endpunkt
            POST /api/minimal-solutions/landingpage_splitter
        


        

            

                Request-Beispiel
                
{
  "topic": "Produkt-Launch",
  "target": "Neukunden",
  "options": "{\"sections\": [\"header\", \"features\"]}"
}

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "topic": "Produkt-Launch",
    "target": "Neukunden",
    "options": "{\"sections\": [\"header\", \"features\"]}",
    "splitter_id": "spl_123456",
    "timestamp": "2026-05-13T12:00:00Z",
    "splitted": true
  }
}
