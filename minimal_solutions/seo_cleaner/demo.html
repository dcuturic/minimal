API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um SEO-Daten basierend auf Topic, Target und Optionen zu bereinigen.

Endpunkt POST /api/minimal-solutions/seo_cleaner
Request-Beispiel 
{
  "topic": "Content Marketing",
  "target": "Beginners",
  "options": {
    "remove_duplicates": true
  }
}

Die Eingabeparameter für die SEO Bereinigung.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "topic": "Content Marketing",
    "target": "Beginners",
    "options": {
      "remove_duplicates": true
    },
    "clean_status": "success",
    "clean_items": 5,
    "message": "Daten für 'Content Marketing' erfolgreich bereinigt."
  }
}
