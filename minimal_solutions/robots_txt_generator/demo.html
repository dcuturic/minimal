            API Documentation
        

        
        

            You can also generate robots.txt programmatically using our REST API. Send a POST request to the endpoint below.
        


        

            

                POST
                /api/minimal-solutions/robots_txt_generator
            

            
        


        

            

                
Request Example

                

                    
{
  "user_agent": "*",
  "allow": "/public/\n/assets/",
  "disallow": "/admin/\n/private/",
  "sitemap_url": "https://example.com/sitemap.xml"
}

                

            

            

                
Response Example

                

                    
{
  "status": "success",
  "data": {
    "result": "User-agent: *\nAllow: /public/\nAllow: /assets/\nDisallow: /admin/\nDisallow: /private/\nSitemap: https://example.com/sitemap.xml"
  }
}
