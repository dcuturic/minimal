Gib eine Dateiendung, einen Dateinamen oder einen MIME Type ein.

API Dokumentation

Du kannst diese Funktion auch über unsere REST-API nutzen. Sende dazu einen POST-Request an den unten stehenden Endpunkt.

POST /api/minimal-solutions/mime_type_lookup

Request Beispiel

{
  "query": ".json"
}

Response Beispiel (Erfolg)

{
  "status": "success",
  "data": {
    "results": [
      {
        "extension": ".json",
        "mime_type": "application/json",
        "description": "JSON format"
      }
    ]
  }
}
