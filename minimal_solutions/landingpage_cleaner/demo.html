API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um Landingpage-Daten programmgesteuert zu bereinigen (Clean).
        


        

            Endpunkt
            POST /api/minimal-solutions/landingpage_cleaner
        


        

            

                Request-Beispiel
                
{
  "topic": "Produkt-Launch",
  "target": "Neukunden",
  "options": {
    "remove_duplicates": true,
    "trim_spaces": true
  }
}

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "topic": "Produkt-Launch",
    "target": "Neukunden",
    "options": {
      "remove_duplicates": true,
      "trim_spaces": true
    },
    "clean_id": "cln_123456",
    "timestamp": "2026-05-13T12:00:00Z",
    "cleaned": true
  }
}
