Provide Base and Target JSON to find missing keys.

Missing Keys

    API Usage

    You can use this tool via our REST API. Send a POST request to the endpoint below.

    POST /api/minimal-solutions/translation_missing_key_finder

    Request Example

    {
  "base_json": {
    "welcome": "Welcome",
    "login": "Sign In"
  },
  "target_json": {
    "welcome": "Willkommen"
  }
}

    Response Example

    {
  "status": "success",
  "data": {
    "missing_keys": [
      "login"
    ]
  }
}
