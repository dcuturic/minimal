Enter value and select units to convert.

API Documentation

You can use this tool programmatically via our API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/einheiten_converter

Request Example

fetch('/api/minimal-solutions/einheiten_converter', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        "value": 100,
        "from_unit": "km",
        "to_unit": "m",
        "category": "length"
    })
});

Response Example

{
    "status": "success",
    "data": {
        "result": 100000.0
    }
}
