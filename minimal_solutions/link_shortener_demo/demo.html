API Documentation

Nutzen Sie die API, um Kurzlinks programmgesteuert zu erstellen.

Endpoint:
POST /api/minimal-solutions/link_shortener_demo
Request Example: 
{
  "long_url": "https://example.com/very/long/url/path",
  "custom_slug": "mein-link"
}
Response Example: 
{
  "status": "success",
  "data": {
    "short_url": "https://short.ly/mein-link",
    "clicks": 0
  }
}
