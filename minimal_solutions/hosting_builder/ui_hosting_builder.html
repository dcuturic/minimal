                API Documentation
            

            

                

                    

                        

                            POST
                            /api/minimal-solutions/hosting_builder
                        

                        
Submit hosting details to trigger a build process.

                        
Request Body (JSON)

                        
    
                            
  • source (string, required): Repository or path.
    • 
                            
  • config (object, optional): Build configuration.
    • 
                            
  • mode (string, optional): Build mode (standard/strict/fast).
    • 
                        

                        
Example Request

                        
{
  "source": "github.com/user/repo",
  "mode": "standard",
  "config": {
    "branch": "main",
    "env": "prod"
  }
}

                    

                    

                        
Example Response (200 OK)

                        
{
  "status": "success",
  "data": {
    "build_id": "bld_123456789",
    "status": "building",
    "source": "github.com/user/repo",
    "mode": "standard"
  }
}
