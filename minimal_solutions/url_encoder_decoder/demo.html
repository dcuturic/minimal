API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert Texte für URLs zu enkodieren und dekodieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/url_encoder_decoder
        


        

            

                Request-Beispiel
                
{
  "mode": "encode",
  "text": "Hello World!"
}

                
Hinweis: `mode` kann "encode" oder "decode" sein.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "result": "Hello%20World%21"
  }
}
