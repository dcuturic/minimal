API Documentation

        
You can use this tool via our API to minify JSON programmatically. Send a POST request to the endpoint below.

        
        

            
POST /api/minimal-solutions/json_minifier

            
            
Request Example:

            
{
  "json_text": "{\n  \"example\": \"data\",\n  \"to\": \"minify\"\n}"
}


            
Response Example:

            
{
  "status": "success",
  "data": {
    "minified_json": "{\"example\":\"data\",\"to\":\"minify\"}",
    "original_size_bytes": 45,
    "minified_size_bytes": 31,
    "saved_percentage": 31.11
  }
}
