API Documentation

        

            Use the API to generate Meta Tags programmatically.
        

        
        

            Endpoint:

            POST /api/minimal-solutions/meta_tag_generator
        

        
        

            

                Request Example:
                
{
  "title": "Example Title",
  "description": "Example Description",
  "image_url": "https://example.com/image.jpg",
  "page_url": "https://example.com"
}

            

            

                Response Example:
                
{
  "status": "success",
  "data": {
    "html": "<title>Example Title</title>\n<meta name=\"description\" content=\"Example Description\">\n..."
  }
}
