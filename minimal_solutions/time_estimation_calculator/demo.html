API Usage

You can use this tool via our REST API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/time_estimation_calculator

Request Example

{
  "ticket_count": 50,
  "minutes_per_ticket": 60,
  "days": 14,
  "workers": 2
}

Response Example

{
  "status": "success",
  "data": {
    "total_hours_required": 50.0,
    "total_hours_available": 224.0,
    "is_feasible": true,
    "utilization_percent": 22.32
  }
}
