API Usage

        

            You can use this tool via our REST API. Send a POST request to the endpoint below.
        

        

            POST /api/minimal-solutions/iframe_preview_url_builder
        

        
        

            

                
Request Example

                
{
  "page": "index",
  "component": "Header",
  "mode": "preview"
}

            

            

                
Response Example

                
{
  "status": "success",
  "data": {
    "preview_url": "/preview?page=index&component=Header&mode=preview",
    "iframe_code": "<iframe src=\"/preview?page=index&component=Header&mode=preview\" ...></iframe>"
  }
}
