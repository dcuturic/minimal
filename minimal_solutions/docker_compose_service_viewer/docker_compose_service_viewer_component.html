API Integration

You can use this tool via API. Send a POST request to /api/minimal-solutions/docker_compose_service_viewer.

Request Example

{
  "compose_yaml": "version: '3'\nservices:\n  web:\n    image: nginx\n    ports:\n      - \"80:80\""
}

Response Example

{
  "status": "success",
  "data": {
    "services": [
      {
        "name": "web",
        "image": "nginx",
        "ports": ["80:80"],
        "volumes": [],
        "build": null
      }
    ]
  }
}
