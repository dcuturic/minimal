Provide source, mode, and config to generate hosting data.

Generated Hosting Configuration


                    

                        
                    

                

            

        

    


    

    
    

        
API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um Hosting-Konfigurationsdaten für ein Projekt zu generieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/hosting_generator
        


        

            

                Request-Beispiel
                
{
  "source": "https://github.com/example/repo",
  "mode": "docker",
  "config": {
    "port": 8080,
    "env": "production"
  }
}

                
Parameter: 
source: Quellcode-Repository oder Verzeichnis
mode: Hosting-Umgebung (docker, static, nodejs)
config: Optionale Parameter als JSON.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "dockerfile": "FROM node:18\n...",
    "commands": [
      "docker build -t app .",
      "docker run -p 8080:8080 app"
    ],
    "environment": "production"
  }
}
