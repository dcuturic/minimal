Enter Component JSON to view dependencies.

Dependencies

API Usage

You can use this tool via our REST API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/component_dependency_viewer

Request Example

{
  "component_json": {
    "name": "MyComponent",
    "dependencies": [
      "Button",
      "Icon"
    ]
  }
}

Response Example

{
  "status": "success",
  "data": {
    "dependencies": [
      "Button",
      "Icon"
    ]
  }
}
