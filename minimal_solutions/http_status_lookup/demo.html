Gib einen HTTP Statuscode ein, um seine Bedeutung zu sehen.

API Dokumentation

Du kannst diese Funktion auch über unsere REST-API nutzen. Sende dazu einen POST-Request an den unten stehenden Endpunkt.

POST /api/minimal-solutions/http_status_lookup

Request Beispiel

{
  "status_code": 404
}

Response Beispiel (Erfolg)

{
  "status": "success",
  "data": {
    "status_code": 404,
    "message": "Not Found",
    "description": "The server can not find the requested resource..."
  }
}
