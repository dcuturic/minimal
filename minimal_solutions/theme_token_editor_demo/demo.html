API Integration

Use this interface externally to validate and parse theme tokens.

Endpoint

POST /api/minimal-solutions/theme_token_editor_demo

Request Example

{
  "tokens": {
    "primary": "#00ffcc",
    "background": "#121212"
  }
}

Response Example

{
  "status": "success",
  "data": {
    "tokens": {
      "primary": "#00ffcc",
      "background": "#121212"
    }
  }
}
