Configure settings to see box-shadow

box-shadow: 10px 10px 20px 0px rgba(0, 0, 0, 0.5);

API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um dynamisch CSS-Code für Schatten-Effekte auf Basis von Parametern zu generieren.

Endpunkt POST /api/minimal-solutions/box_shadow_generator
Request-Beispiel 
{
  "x": 10,
  "y": 10,
  "blur": 20,
  "spread": 0,
  "opacity": 0.5,
  "color": "#000000",
  "inset": false
}

Parameter für den Schatten (Offsets, Blur, Spread, etc.).

Response-Beispiel 
{
  "status": "success",
  "data": {
    "css": "box-shadow: 10px 10px 20px 0px rgba(0, 0, 0, 0.5);"
  }
}
