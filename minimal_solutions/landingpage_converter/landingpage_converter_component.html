API Integration

You can use this tool via our REST API. Send a POST request with the topic, target, and options to convert the data.

Endpoint POST /api/minimal-solutions/landingpage_converter
Request Example 
{
  "topic": "Modern AI SaaS",
  "target": "Developers and Tech Founders",
  "options": "Dark mode, easy integration, low latency"
}
Response Example 
{
  "status": "success",
  "data": {
    "converted_data": {
      "headline": "...",
      "subheadline": "...",
      "features": ["..."]
    },
    "meta": {
      "topic": "Modern AI SaaS",
      "target": "Developers and Tech Founders"
    }
  }
}
