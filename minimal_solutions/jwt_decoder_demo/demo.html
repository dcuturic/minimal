Gib einen JWT ein, um Header und Payload zu sehen.

API Dokumentation

Du kannst diese Funktion auch über unsere REST-API nutzen. Sende dazu einen POST-Request an den unten stehenden Endpunkt.

POST /api/minimal-solutions/jwt_decoder_demo

Request Beispiel

{
  "jwt": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c"
}

Response Beispiel (Erfolg)

{
  "status": "success",
  "data": {
    "header": {
      "alg": "HS256",
      "typ": "JWT"
    },
    "payload": {
      "sub": "1234567890",
      "name": "John Doe",
      "iat": 1516239022
    }
  }
}
