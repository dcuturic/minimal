API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert Component JSONs aus Template, Env, CSS und JS zusammenzubauen.
        


        

            Endpunkt
            POST /api/minimal-solutions/component_json_builder
        


        

            

                Request-Beispiel
                
{
  "template": "<div class=\"my-component\">\n  Hello {{ name }}\n</div>",
  "env": { "name": "World" },
  "css": ".my-component { color: red; }",
  "js": "console.log('Component initialized');"
}

                
Hinweis: `env`, `css` und `js` sind optional, `template` ist erforderlich.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "template": "<div class=\"my-component\">\n  Hello {{ name }}\n</div>",
    "env": {
      "name": "World"
    },
    "css": ".my-component { color: red; }",
    "js": "console.log('Component initialized');"
  }
}
