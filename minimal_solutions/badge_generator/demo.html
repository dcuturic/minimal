API Documentation

The Badge Generator functionality can also be accessed via API.

Endpoint POST /api/minimal-solutions/badge_generator
Request Example 
{
  "label": "build",
  "value": "passing",
  "style": "flat"
}
Response Example 
{
  "status": "success",
  "data": {
    "badge_svg": "<svg...>...</svg>"
  }
}
