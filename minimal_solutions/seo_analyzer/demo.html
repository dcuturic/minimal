API-Schnittstelle

Der SEO Analyzer kann per API angesprochen werden. Hier ist ein Beispiel für Request und Response.

Endpunkt POST /api/minimal-solutions/seo_analyzer
Request-Beispiel 
{
  "topic": "https://example.com",
  "target": "Desktop",
  "options": ["check_density"]
}
Response-Beispiel 
{
  "status": "success",
  "data": {
    "score": 85,
    "issues": [],
    "keyword_density": "2.5%"
  }
}
