API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert JSON-Strings zu formatieren oder zu minifizieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/json_formatter
        


        

            

                Request-Beispiel
                
{
  "json_text": "{\"key\":\"value\"}",
  "indent": 2
}

                
Hinweis: `indent` kann eine Zahl (z.B. 2, 4) oder 0 sein, um das JSON zu minifizieren.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "formatted_json": "{\n  \"key\": \"value\"\n}"
  }
}
