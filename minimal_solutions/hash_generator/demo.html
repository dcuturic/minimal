API Documentation

        
The Hash Generator functionality can also be accessed via API.

        
        

            Endpoint
            POST /api/minimal-solutions/hash_generator
        

        
        

            Request Example
            
{
  "text": "hello world",
  "algorithm": "sha256"
}

        

        
        

            Response Example
            
{
  "status": "success",
  "data": {
    "result": "b94d27b9934d3e08a52e52d7da7dabfac484efe37a5380ee9088f7ace2efcde9",
    "algorithm": "sha256"
  }
}
