API Documentation

Use the API to generate OpenGraph preview data programmatically.

Endpoint:
POST /api/minimal-solutions/opengraph_preview
Request Example: 
{
  "title": "My Awesome Website",
  "description": "This is an awesome website about cool things.",
  "image_url": "https://example.com/image.png",
  "domain": "example.com"
}
Response Example: 
{
  "status": "success",
  "data": {
    "title": "My Awesome Website",
    "description": "This is an awesome website about cool things.",
    "image_url": "https://example.com/image.png",
    "domain": "example.com"
  }
}
