Enter value and units to convert.

API Documentation

You can use this tool programmatically via our API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/dateigroessen_converter

Request Example

fetch('/api/minimal-solutions/dateigroessen_converter', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        "value": 1024,
        "from_unit": "MB",
        "to_unit": "GB"
    })
});

Response Example

{
    "status": "success",
    "data": {
        "result": 1.024,
        "formatted": "1.02 GB"
    }
}
