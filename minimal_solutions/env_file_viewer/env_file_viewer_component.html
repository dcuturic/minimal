API Integration

You can use this tool via API. Send a POST request to /api/minimal-solutions/env_file_viewer.

Request Example

{
  "env_text": "DB_HOST=localhost\nDB_PASSWORD=secret123\nAPI_KEY=sk_live_abc123"
}

Response Example

{
  "status": "success",
  "data": {
    "env_vars": [
      {
        "key": "DB_HOST",
        "value": "localhost"
      },
      {
        "key": "DB_PASSWORD",
        "value": "secret123"
      },
      {
        "key": "API_KEY",
        "value": "sk_live_abc123"
      }
    ]
  }
}
