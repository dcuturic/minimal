API Documentation

        

            Use the API to generate Sitemap XML programmatically.
        

        
        

            Endpoint:

            POST /api/minimal-solutions/sitemap_xml_generator
        

        
        

            

                Request Example:
                
{
  "urls": [
    "https://example.com/",
    "https://example.com/about/"
  ],
  "changefreq": "daily",
  "priority": 0.5
}

            

            

                Response Example:
                
{
  "status": "success",
  "data": {
    "sitemap_xml": "<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n<urlset xmlns=\"http://www.sitemaps.org/schemas/sitemap/0.9\">\n..."
  }
}
