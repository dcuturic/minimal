API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Templates für SEO-Daten zu bauen.

Endpunkt POST /api/minimal-solutions/seo_template_builder
Request-Beispiel 
{
  "topic": "E-commerce SEO Strategy",
  "target": "Online Store Owners",
  "options": {
    "format": "json"
  }
}

Übergibt Topic, Target und Konfigurationsoptionen.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "topic": "E-commerce SEO Strategy",
    "target": "Online Store Owners",
    "template_status": "success",
    "message": "Template für 'E-commerce SEO Strategy' erfolgreich generiert.",
    "options_applied": {
      "format": "json"
    }
  }
}
