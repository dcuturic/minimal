Formatted Notes

API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um unstrukturierte Notizen automatisch in formatierte Listen und Action Items zu konvertieren.

Endpunkt POST /api/minimal-solutions/meeting_notes_formatter
Request-Beispiel 
{
  "notes": "team sync. talked about budget and q3 goals. anna will prepare the presentation by friday. budget approved for 50k. need to call the agency."
}

Hinweis: `notes` ist ein String, der nicht leer sein darf.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "formatted_notes": "- Team Sync.\n- Talked about budget and Q3 goals.\n- Budget approved for 50k.",
    "action_items": [
      "Anna will prepare the presentation by friday.",
      "Need to call the agency."
    ]
  }
}
