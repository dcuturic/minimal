Daten eingeben, um Call-to-Action Vorschläge zu generieren.

API Dokumentation

Diese Minimal-Lösung kann auch direkt als externe API über POST-Requests verwendet werden. Senden Sie einfach die Parameter als JSON an den Endpunkt.

Endpoint POST /api/minimal-solutions/cta_generator

Request-Beispiel (JSON)

{
  "topic": "Neuer Sommer-Sale",
  "audience": "Junge Erwachsene",
  "tone": "enthusiastic"
}

Response-Beispiel (JSON)

{
  "status": "success",
  "data": {
    "suggestions": [
      "Shoppe jetzt den Sommer-Sale!",
      "Sichere dir die besten Angebote für junge Erwachsene!"
    ]
  }
}
