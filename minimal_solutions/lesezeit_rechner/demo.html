API Documentation

Du kannst diese Minimal-Lösung auch über unsere API in deine eigenen Anwendungen integrieren.

Endpoint: POST /api/minimal-solutions/lesezeit_rechner
Request Example: 
{
  "text": "Dieser Text wird berechnet...",
  "words_per_minute": 200
}
Response Example: 
{
  "status": "success",
  "data": {
    "result": "1 min 0 sek",
    "minutes": 1,
    "seconds": 0,
    "total_seconds": 60,
    "word_count": 200
  }
}
