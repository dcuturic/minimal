API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Minecraft-Daten basierend auf input_text, mode und weiteren Optionen zu vergleichen (diff).

Endpunkt POST /api/minimal-solutions/minecraft_diff_viewer
Request-Beispiel 
{
  "input_text": [{"name": "Diamond Sword"}, {"name": "Iron Sword"}],
  "mode": "diff",
  "options": {
    "strict_mode": true
  }
}

Die Felder input_text, mode und options sind erforderlich (bzw. gemäss Validierung).

Response-Beispiel 
{
  "success": true,
  "data": {
    "diff_result": [
      {"name": "Diamond Sword"},
      {"name": "Iron Sword"}
    ],
    "mode": "diff",
    "options": {
      "strict_mode": true
    },
    "message": "Diff erfolgreich durchgeführt."
  }
}
