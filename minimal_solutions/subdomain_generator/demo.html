Enter name and base domain, then click generate.

    API Documentation

    You can use this tool via our REST API. Send a POST request to the endpoint with your name/keyword and base domain.

    POST /api/minimal-solutions/subdomain_generator

    Request Example

    {
  "name": "shop",
  "base_domain": "example.com"
}

    Response Example

    {
  "status": "success",
  "data": {
    "subdomains": [
      "shop.example.com",
      "my-shop.example.com",
      "get-shop.example.com",
      "shop-app.example.com"
    ]
  }
}
