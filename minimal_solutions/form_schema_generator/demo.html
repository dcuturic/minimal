Define fields and click generate.

API Integration

You can use this tool programmatically via our API. Send a POST request with the required fields to generate a JSON Schema.

Endpoint
POST /api/minimal-solutions/form_schema_generator
Request Example
{
  "fields": [
    {
      "name": "username",
      "type": "string",
      "required": true
    },
    {
      "name": "age",
      "type": "integer",
      "required": false
    }
  ]
}
Response Example
{
  "status": "success",
  "data": {
    "schema": {
      "type": "object",
      "properties": {
        "username": {
          "type": "string"
        },
        "age": {
          "type": "integer"
        }
      },
      "required": [
        "username"
      ]
    }
  }
}
