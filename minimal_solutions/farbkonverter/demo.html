API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Farbwerte zwischen verschiedenen Formaten zu konvertieren.

Endpunkt POST /api/minimal-solutions/farbkonverter
Request-Beispiel 
{
  "color_value": "#FF5733"
}

Hinweis: `color_value` kann als HEX, RGB, RGBA oder HSL übergeben werden.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "hex": "#FF5733",
    "rgb": "rgb(255, 87, 51)",
    "rgba": "rgba(255, 87, 51, 1)",
    "hsl": "hsl(11, 100%, 60%)"
  }
}
