API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um programmgesteuert Timestamps in menschenlesbare Daten umzuwandeln und umgekehrt.

Endpunkt POST /api/minimal-solutions/timestamp_converter
Request-Beispiel 
{
  "value": "1704067200",
  "mode": "timestamp_to_date",
  "timezone": "UTC"
}

Hinweis: `mode` kann "timestamp_to_date" oder "date_to_timestamp" sein. `timezone` kann "UTC" oder "Local" sein.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "result": "2024-01-01 00:00:00"
  }
}
