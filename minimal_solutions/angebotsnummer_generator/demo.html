Pattern: [Prefix]-[Date]-[Customer]-[Number]

API Documentation

Use this endpoint to generate customized offer numbers systematically via REST API.

POST /api/minimal-solutions/angebotsnummer_generator

Request

{
  "prefix": "ANG",
  "date": "2026-05",
  "customer_short": "CUST",
  "number": 1
}

Response

{
  "status": "success",
  "data": {
    "numbers": [
      "ANG-2026-05-CUST-0001"
    ],
    "pattern": "ANG-2026-05-CUST-0001"
  }
}
