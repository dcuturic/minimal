Select mode and count, then click Generate.

API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um programmgesteuert Platzhaltertexte (Lorem Ipsum) zu generieren.

Endpunkt POST /api/minimal-solutions/lorem_ipsum_generator
Request-Beispiel 
{
  "mode": "paragraphs",
  "count": 2
}

Hinweis: `mode` (paragraphs, words, bytes, lists), `count` (1-100).

Response-Beispiel 
{
  "status": "success",
  "data": {
    "text": "Lorem ipsum dolor sit amet..."
  }
}
