API-Schnittstelle

Der SEO Summarizer kann per API angesprochen werden. Hier ist ein Beispiel für Request und Response.

Endpunkt POST /api/minimal-solutions/seo_summarizer
Request-Beispiel 
{
  "topic": "https://example.com/blog-post",
  "target": "Management Summary",
  "options": ["extract_keywords"]
}
Response-Beispiel 
{
  "status": "success",
  "data": {
    "summary": "Die Seite behandelt...",
    "key_findings": ["Punkt 1", "Punkt 2"],
    "extracted_keywords": ["SEO", "Summary"]
  }
}
