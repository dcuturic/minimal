Enter source text and desired count to generate FAQs.

API Documentation

You can also use this tool via our REST API. Send a POST request to the endpoint below.

Endpoint POST /api/minimal-solutions/faq_generator
Example Request 
{
  "source_text": "Our new service offers 24/7 support and unlimited bandwidth. It costs $9.99 per month. You can cancel anytime.",
  "count": 3
}
Example Response 
{
  "status": "success",
  "data": {
    "faqs": [
      {
        "question": "What kind of support is available?",
        "answer": "We offer 24/7 support."
      },
      {
        "question": "How much does the service cost?",
        "answer": "The service costs $9.99 per month."
      },
      {
        "question": "Can I cancel my subscription?",
        "answer": "Yes, you can cancel anytime."
      }
    ]
  }
}
