API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert Texte in Base64 zu konvertieren und zurück.
        


        

            Endpunkt
            POST /api/minimal-solutions/base64_encoder_decoder
        


        

            

                Request-Beispiel
                
{
  "mode": "encode",
  "text": "Hello World"
}

                
Hinweis: `mode` kann "encode" oder "decode" sein.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "result": "SGVsbG8gV29ybGQ="
  }
}
