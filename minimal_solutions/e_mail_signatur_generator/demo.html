API Documentation

You can use this tool via our API to generate email signatures programmatically.

POST /api/minimal-solutions/e_mail_signatur_generator
Request Example:
{
  "name": "Max Mustermann",
  "role": "Software Engineer",
  "company": "CraftHoster",
  "email": "max@example.com",
  "phone": "+49 123 456789",
  "website": "https://crafthoster.de"
}
Response Example:
{
  "status": "success",
  "data": {
    "signature_html": "<div>...</div>"
  }
}
