API Dokumentation

Diese Minimal-Lösung ist auch als REST-API verfügbar. Du kannst Anfragen direkt an den Server senden und erhältst die Antwort im JSON-Format.

Endpunkt

POST /api/minimal-solutions/countdown_generator

Request (JSON)

{
  "target_date": "2027-01-01T00:00:00",
  "label": "Neujahr"
}

Response (JSON)

{
  "status": "success",
  "data": {
    "title": "Neujahr",
    "days": 234,
    "hours": 5,
    "minutes": 34,
    "seconds": 8
  }
}
