API & Integration

Die CSS Gradient Generator Schnittstelle steht auch als REST-API zur Verfügung. Sende einen POST-Request mit den gewünschten Konfigurationen, um den fertigen CSS-Code als Antwort zu erhalten.

API Endpoint

POST /api/minimal-solutions/css_gradient_generator

Request Beispiel

{
  "type": "linear",
  "colors": ["#FF5733", "#3333FF"],
  "angle": 90
}

Response Beispiel

{
  "status": "success",
  "data": {
    "css": "linear-gradient(90deg, #FF5733, #3333FF)"
  }
}
