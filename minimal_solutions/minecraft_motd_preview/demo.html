API Documentation

You can use this tool via our REST API. Send a POST request to the endpoint with your MOTD text.

POST /api/minimal-solutions/minecraft_motd_preview

Request Example

{
  "motd_text": "&aWelcome to our\\n&cAwesome Server"
}

Response Example

{
  "status": "success",
  "data": {
    "motd_text": "&aWelcome to our\\n&cAwesome Server"
  }
}
