Hinweis: Dieses Tool erstellt beispielhafte Datenschutz-Bausteine und stellt keine Rechtsberatung dar. Die generierten Texte müssen vor Verwendung geprüft werden.

Wählen Sie die Optionen aus und klicken Sie auf Generieren.

API Referenz

Diese Minimal-Lösung kann auch über unsere REST-API integriert werden.

Endpunkt POST /api/minimal-solutions/datenschutz_text_demo
Request (JSON) 
{
  "uses_cookies": true,
  "uses_analytics": true,
  "uses_contact_form": false,
  "hosting_provider": "Hetzner Online GmbH"
}
Response (JSON) 
{
  "status": "success",
  "data": {
    "text": "1. Datenschutz auf einen Blick\n..."
  }
}
