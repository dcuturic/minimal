API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um programmgesteuert Status-Page Incident-Updates zu generieren.

Endpunkt POST /api/minimal-solutions/status_page_incident_generator
Request-Beispiel 
{
  "service": "Core Platform",
  "incident_type": "Outage",
  "status": "Investigating",
  "message": "We are currently investigating..."
}

Hinweis: Gültige Werte für `incident_type` sind "Outage", "Degraded Performance", "Scheduled Maintenance". Gültige Werte für `status` sind "Investigating", "Identified", "Monitoring", "Resolved".

Response-Beispiel 
{
  "status": "success",
  "data": {
    "result": {
      "incident": {
        "service": "Core Platform",
        "incident_type": "Outage",
        "status": "Investigating",
        "message": "We are currently investigating...",
        "created_at": "2026-05-12T14:19:14Z",
        "id": "inc_1234567890"
      },
      "formatted_text": "[2026-05-12T14:19:14Z] INCIDENT..."
    }
  }
}
