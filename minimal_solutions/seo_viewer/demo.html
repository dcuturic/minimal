API-Schnittstelle

Der SEO Viewer kann per API angesprochen werden. Hier ist ein Beispiel für Request und Response.

Endpunkt POST /api/minimal-solutions/seo_viewer
Request-Beispiel 
{
  "topic": "https://example.com",
  "target": "Desktop",
  "options": ["extract_meta"]
}
Response-Beispiel 
{
  "status": "success",
  "data": {
    "title": "Example Domain",
    "description": "An example domain for illustrations.",
    "keywords": ["example", "domain"]
  }
}
