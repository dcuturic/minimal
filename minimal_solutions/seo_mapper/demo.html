API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um SEO-Daten basierend auf einem Topic und Target zu mappen.

Endpunkt POST /api/minimal-solutions/seo_mapper
Request-Beispiel 
{
  "topic": "Content Marketing",
  "target": "increase_traffic",
  "options": "fast_mode"
}

Parameter für das Mappen der SEO-Daten.

Response-Beispiel 
{
  "status": "success",
  "data": {
    "topic": "Content Marketing",
    "target": "increase_traffic",
    "mapped_status": "success",
    "message": "Daten für 'Content Marketing' erfolgreich gemappt."
  }
}
