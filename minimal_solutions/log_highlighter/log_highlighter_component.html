Paste logs and click Highlight Logs.

API Documentation

You can also use this tool via our REST API. Send a POST request to the endpoint below.

Endpoint POST /api/minimal-solutions/log_highlighter
Example Request 
{
  "log_text": "2023-10-27 10:00:00 INFO  App started\n2023-10-27 10:00:01 ERROR Database disconnected"
}
Example Response 
{
  "status": "success",
  "data": {
    "html": "<span class=\"log-line log-level-info\">...</span>",
    "lines": [
      {
        "text": "2023-10-27 10:00:00 INFO  App started\n",
        "level": "INFO"
      }
    ]
  }
}
