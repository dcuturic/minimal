API Documentation

Generate Docker Container Cards via API.

Endpoint POST /api/minimal-solutions/docker_container_card_demo
Request Example 
{
  "name": "web-server",
  "status": "running",
  "image": "nginx:latest",
  "ports": "80:80"
}
Response Example 
{
  "status": "success",
  "data": {
    "name": "web-server",
    "status": "running",
    "image": "nginx:latest",
    "ports": "80:80",
    "timestamp": "2026-05-12T10:14:00Z"
  }
}
