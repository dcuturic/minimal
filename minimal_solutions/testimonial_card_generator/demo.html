API-Dokumentation

Diese Minimal-Lösung kann auch direkt als API in externe Anwendungen integriert werden.

POST /api/minimal-solutions/testimonial_card_generator

Request-Beispiel (JSON)
{
  "name": "Jane Doe",
  "role": "CEO at TechCorp",
  "quote": "This product changed our workflow completely!"
}
Response-Beispiel (JSON)
{
  "status": "success",
  "data": {
    "name": "Jane Doe",
    "role": "CEO at TechCorp",
    "quote": "This product changed our workflow completely!",
    "html": "<div class=\"tc-result-card\"...>...</div>",
    "css": ".tc-result-card { ... }"
  }
}
