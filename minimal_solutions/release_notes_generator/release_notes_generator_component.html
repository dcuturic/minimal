API Integration

        
You can use this tool via API. Send a POST request to /api/minimal-solutions/release_notes_generator.

        
        

            

                
Request Example

                
{
  "version": "2.1.0",
  "features": [
    "New dashboard widget",
    "Support for dark mode"
  ],
  "fixes": [
    "Fixed login timeout issue"
  ],
  "breaking_changes": []
}

            

            

                
Response Example

                
{
  "status": "success",
  "data": {
    "result": "## Release Notes v2.1.0\n\n### Features\n- New dashboard widget\n- Support for dark mode\n\n### Fixes\n- Fixed login timeout issue\n"
  }
}
