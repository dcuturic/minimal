API / Schnittstelle

Dieser Service kann extern als API genutzt werden, um Minecraft-Daten basierend auf input_text, mode und weiteren Optionen zu mappen.

Endpunkt POST /api/minimal-solutions/minecraft_mapper
Request-Beispiel 
{
  "input_text": "Diamond Sword",
  "mode": "map",
  "options": {
    "map_properties": true
  }
}

Die Felder input_text, mode und options sind erforderlich (bzw. gemäss Validierung).

Response-Beispiel 
{
  "success": true,
  "data": {
    "mapped_text": "Diamond Sword",
    "mode": "map",
    "options": {
      "map_properties": true
    },
    "message": "Mapping erfolgreich durchgeführt."
  }
}
