API Documentation

You can use this tool programmatically via our API. Send a POST request to the endpoint below.

POST /api/minimal-solutions/word_replacer

Request Example

fetch('/api/minimal-solutions/word_replacer', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        "text": "Hello world, hello universe!",
        "search": "hello",
        "replace": "Hi",
        "case_sensitive": false
    })
});

Response Example

{
    "status": "success",
    "data": {
        "result_text": "Hi world, Hi universe!",
        "replacement_count": 2
    }
}
