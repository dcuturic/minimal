API Dokumentation

Diese Minimal-Lösung kann auch über eine REST-Schnittstelle von externen Anwendungen genutzt werden. Senden Sie einen POST-Request an den Endpunkt mit dem Markdown-Text.

Endpoint:
POST /api/minimal-solutions/markdown_preview
Request (JSON): 
{
  "markdown_text": "# Hello\n\n**World**"
}
Response (JSON): 
{
  "status": "success",
  "data": {
    "html": "<h1>Hello</h1>\n<p><strong>World</strong></p>"
  }
}
