API Dokumentation

            
Endpoint: POST /api/minimal-solutions/hosting_exporter

            
Die Schnittstelle kann von externen Systemen aufgerufen werden, indem ein POST-Request mit den entsprechenden Parametern (source, config, mode) als JSON-Body gesendet wird.

            
Request-Beispiel:

            
{
  "source": "{\"host\": \"example.com\", \"export_enabled\": true}",
  "config": "{\"strict\": true}",
  "mode": "quick"
}
            

            
Response-Beispiel:

            
{
  "success": true,
  "data": {
    "source": "{\"host\": \"example.com\", \"export_enabled\": true}",
    "config": "{\"strict\": true}",
    "mode": "quick",
    "export_result": "...",
    "message": "Hosting successfully exported."
  },
  "error": null,
  "warnings": [],
  "meta": {}
}
