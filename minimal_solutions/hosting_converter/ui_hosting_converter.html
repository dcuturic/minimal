API Dokumentation

            
Endpoint: POST /api/minimal-solutions/hosting_converter

            
Die Schnittstelle kann von externen Systemen aufgerufen werden, indem ein POST-Request mit den entsprechenden Parametern (source, config, mode) als JSON-Body gesendet wird.

            
Request-Beispiel:

            
{
  "source": "{\"host\": \"example.com\", \"type\": \"shared\"}",
  "config": "{}",
  "mode": "json"
}
            

            
Response-Beispiel:

            
{
  "success": true,
  "data": {
    "source": "{\"host\": \"example.com\", \"type\": \"shared\"}",
    "config": "{}",
    "mode": "json",
    "converted_data": "...",
    "message": "Hosting successfully converted."
  },
  "error": null,
  "warnings": [],
  "meta": {}
}
