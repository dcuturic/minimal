API Documentation

        

            You can use this tool programmatically via our API. Send a POST request to the endpoint below.
        

        
        

            

                POST
                /api/minimal-solutions/case_converter
            

            
            

                

                    
Request Example

                    
fetch('/api/minimal-solutions/case_converter', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        "text": "Hello world",
        "target_case": "uppercase"
    })
});

                

                

                    
Response Example

                    
{
    "status": "success",
    "data": {
        "result": "HELLO WORLD"
    }
}
