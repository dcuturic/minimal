Configure details to generate a backup plan.


                

            

        

    


    

    
    

        
API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert Backup-Pläne zu generieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/backup_plan_generator
        


        

            

                Request-Beispiel
                
{
  "frequency": "daily",
  "retention_days": 30,
  "storage_type": "s3"
}

                
Hinweis: `frequency` kann "hourly", "daily", "weekly" oder "monthly" sein. `storage_type` unterstützt "local", "s3", "azure", "gcs".

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "plan_id": "bp_123456789",
    "frequency": "daily",
    "retention_days": 30,
    "storage_type": "s3",
    "next_run": "2026-05-13T00:00:00Z",
    "estimated_cost_per_gb": 0.023,
    "actions": [
      "Snapshot EBS volumes",
      "Sync to S3 bucket"
    ]
  }
}
