Configure details to generate the maintenance page.


                

            

        

    


    
    
    

        
API / Schnittstelle

        

            Dieser Service kann extern als API genutzt werden, um programmgesteuert Maintenance-Seiten (Wartungsseiten) zu generieren.
        


        

            Endpunkt
            POST /api/minimal-solutions/maintenance_page_generator
        


        

            

                Request-Beispiel
                
{
  "title": "Under Maintenance",
  "message": "We are currently performing scheduled maintenance.",
  "eta": "2 Hours"
}

                
Hinweis: `title` und `message` sind Pflichtfelder. `eta` ist optional.

            

            

                Response-Beispiel
                
{
  "status": "success",
  "data": {
    "html_output": "<!DOCTYPE html>\n<html lang=\"en\">\n<head>..."
  }
}
