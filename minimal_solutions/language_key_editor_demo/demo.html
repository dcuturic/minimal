Provide keys and languages to generate the editor.

API Integration

Use this interface externally to generate language key editor translations.

Endpoint

POST /api/minimal-solutions/language_key_editor_demo

Request Example

{
  "keys": [
    "login.title",
    "login.submit"
  ],
  "languages": [
    "en",
    "de"
  ]
}

Response Example

{
  "status": "success",
  "data": {
    "translations": {
      "login.title": {
        "en": "",
        "de": ""
      },
      "login.submit": {
        "en": "",
        "de": ""
      }
    }
  }
}
