Passe die Parameter an und klicke auf generieren.

API-Dokumentation

Verwende die REST-API, um Favicons extern zu generieren.

Endpoint:
POST /api/minimal-solutions/favicon_generator_demo
Request-Beispiel: 
{
  "letters": "CH",
  "primary_color": "#3b82f6",
  "shape": "rounded"
}
Response-Beispiel: 
{
  "status": "success",
  "data": {
    "favicon_svg": "<svg xmlns=\"http://www.w3.org/2000/svg\"...>...</svg>"
  }
}
