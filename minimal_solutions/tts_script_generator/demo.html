API Integration

You can use this tool via our REST API. Send a POST request with the topic, voice_style, and duration.

Endpoint POST /api/minimal-solutions/tts_script_generator
Request Example 
{
  "topic": "Benefits of AI",
  "voice_style": "professional",
  "duration": "30s"
}
