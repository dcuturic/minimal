API Integration

        
Du kannst diese Minimal-Lösung auch per API ansprechen.

        
        

            POST /api/minimal-solutions/json_zu_csv_converter
        


        
Request Beispiel

        
{
  "json_text": "[\n  {\n    \"name\": \"Max\",\n    \"age\": 30\n  }\n]",
  "delimiter": ","
}


        
Response Beispiel

        
{
  "status": "success",
  "data": {
    "csv_text": "name,age\nMax,30"
  }
}
