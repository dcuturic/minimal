API Documentation

The Server Resource Card Generator functionality can also be accessed via API.

Endpoint POST /api/minimal-solutions/server_resource_card_demo
Request Example 
{
  "cpu": 45.5,
  "ram": 78.2,
  "disk": 92.0
}
Response Example 
{
  "status": "success",
  "data": {
    "cpu": 45.5,
    "ram": 78.2,
    "disk": 92.0,
    "timestamp": "2026-05-12T09:50:36Z"
  }
}
